表单
	作用
		1.给用户提供可视化的控件
		2.收集用户信息提交给服务器
	语法
		表单
			标记
				<form></form>
			属性
				action
					提交的服务器地址，默认是提交给本页面
				method
					数据提交的方式
					取值
						get
							1.明文提交，提交的数据会直接显示到地址栏上去
							2.提交数据大小限制为2Kb
							3.安全性低
							4.当需要从服务器要数据时用get
						post
                            1.密文提交（隐式提交）
                            2.提交数据大小无限制
                            3.安全性高
                            4.向服务器发送数据用post
                        put
                        delete
    表单控件
        能够与用户进行交互的元素
         分类
            1.input 元素
                作用
                    给用户提供各种输入控件，文本框、密码框、单选按钮、复选按钮
                语法
                    标记
                        <input>或<input />
                    属性
                        type
                            定义表单控件的类型
                        name
                            给表单控件起名字，服务器使用
                        value
                            控件的值
                        disabled
                            禁止使用，不能提交数据，也不能操作，默认无值，只要出现这个属性，就禁用
                    详解
                        文本框和密码框
                            <input type="text" /> 默认类型
                            <input tepe="password" /> 密码框
                            属性
                                    placeholder
                                    占位符，表单控件中默认出现的文本
                                maxlength
                                    指定控件输入的字符数
                                readonly
                                    只读， 不能改，只能看，允许将数据提交给服务器
                        按钮
                            普通按钮
                                <input type="button"> 没有功能
                            提交按钮
                                <input type="submit"> 将表单中的数据提交给服务器（action指定的）
                            重置按钮
                                <input type="reset"> 将表单中的内容恢复到初始化状态
                            属性
                                value
                                    定义按钮上的文字
                        单选按钮和复选框
                            <input type="radio" >
                            <input type="checkbox" >
                            属性
                                name
                                    定义控件名称的同时进行分组
                                checked
                                    没有属性值，默认被选中
                        文件选择框
                            <input type="file" >
                            属性
                                method
                                    提交方式，必须是post
                                enctype
                                    multipart/form-data
            2.textarea 多行文本元素
                作用
                    录入多行数据的文本框
                语法
                    标记
                        <textarea></textarea>
                    属性
                        name
                            给控件起名字，提供给服务器使用
                        readonly
                            只读属性
                        cols
                            指定文本框的列数，指定一行中可以输入的英文字符数（中文减半）,出现滚动条的情况下
                        rows
                            指定文本框的行数，即默认显示的行数，如果输入的内容超过指定行数会出现滚动条
            3.select和option 元素
                选项框
                    语法
                        标记
                            <select></select>
                        属性
                            size
                            name
                            multiple
                选项
                    语法
                        标记
                            <option>内容</option>
                        属性
                            value
                                定义选项的值
                            selected
                                指定默认选中项

            4.其他元素
                label
                    作用
                        关联文本和表单控件
                    语法
                        标记
                            <label></label>
                        属性
                            for
                                值：与文本关联的表单控件的id值
